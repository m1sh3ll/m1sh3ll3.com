# m1sh3ll3.com

The source for [m1sh3ll3.com](https://m1sh3ll3.com) — my personal portfolio site, built with Blazor Server.

## About

I'm Michelle Nesbitt, a C#/.NET developer with an e-commerce operations background. This site is where I showcase the internal tools I've built on the job and on my own — desktop apps, web apps, and everything in between.

It also hosts **Project Unify**, a self-directed packaging framework where I present years of real, already-built tooling as one coherent portfolio. Every app shown on the site is real software, built for actual employers or personal projects — there is no "Plugzilla Global," that's just the fictional umbrella I use to frame the case study.

## Tech Stack

- C# / .NET
- Blazor Server
- HTML / CSS / JavaScript (project detail pages under `wwwroot/apps`)

## Project Structure

```
Pages/          Razor pages (home page, layout routing)
wwwroot/        Static assets — styles, scripts, project detail pages, screenshots
Program.cs      App startup/configuration
```

## Running Locally

```
dotnet run
```

Then open the URL shown in the console (typically `https://localhost:5001`).
